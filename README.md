## Admin Dashboard

Web

```bash
 https://firmansyf.github.io/admin-dashboard
```

Step by step run project

```bash
 git clone https://github.com/firmansyf/admin-dashboard.git

 npm install or yarn install

 yarn dev
```

### Main Feature

- Create Data
- Read Data
- Update Data
- Delete Data
- Searching Data By. Name

### Stack

- Vite

- Tailwind
- Apex Chart
